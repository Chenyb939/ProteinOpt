    ProteinOpt: Protein Stability Optimization Tool

To commence utilizing ProteinOpt, navigate to the homepage and specify the optimization technique of your choice to access the task submission interface.

Point Mutation Scanning

    Point mutation scanning is used to calculate the energy of the mutated protein to find the amino acids that are essential to the stability of the protein. To execute this method, input a Job Name to track your task, upload the relevant PDB file, select the specific protein chain for enhancement, determine the optimal number of seed residues, allocate computational threads, and set the parameters for the generation of result files.

RosettaVIP

    RosettaVIP is a method that implements the RosettaVIP protocol to identify amino acids that can effectively fill cavities within protein structures. For its utilization, it is imperative to specify a unique Job Name for task delineation, submit the pertinent PDB file containing the protein structure, select the target protein chain for optimization, designate the number of computational threads for processing, and indicate the desired number of resultant output files.

Supercharge

    Supercharge identifies polar amino acids on proteins to use as seed residues, aiming to change the protein's electrostatic potential by controlling the mutations of these residues. Users must provide a Job Name to track their task, upload the PDB file, choose the protein chain for optimization, set the optimization target and target potential, select the number of threads for processing, and indicate how many result files should be produced.

Manually Specificed

    Manually Specified directly mutates the amino acids specified by the user and uses them as seed residues to further optimize the protein's stability. Users must input a Job Name to identify their task, upload the PDB file, choose the protein chain to optimize, list the amino acid mutations separated by commas, set the number of threads for computation, and indicate the number of result files to generate.

Notice

    Please verify that the protein chain you are optimizing is included in your protein file. Although using more threads can speed up the optimization, the acceleration is not a linear relationship. Keep in mind that because the Rosetta optimization process involves randomness, there may be some repetition in the results produced.
